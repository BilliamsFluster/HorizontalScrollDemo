About Smooth Scroll

Smooth Scrolling Demo

GSAP is framework agnostic and can be used in React, Webflow, Wordpress, or any other JS/Web frameworks. The core GSAP file and all the plugins are just Javascript files. This install helper covers the most common ways to load the files. Namely via NPM, Yarn, and using a simple tag. Pick your own adventure or check out our install guides for framework or tool specific guidance.
